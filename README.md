1. Tokenization
2. Stemming
3. Lemmatization
4. Splitting
5. Bag-of-words
6. Text classifier
7. Gender classifier
8. Sentiment analysis

Per ciascun argomento:
 - verranno scritti uno o più file sorgente .py, tutti condivisi in questo repository;
 - verranno realizzati uno o più blog post, pubblicati su www.lucanaso.it/blog/.
